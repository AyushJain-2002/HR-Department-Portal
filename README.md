 
This project is an Insurance Portal CRM/ERP system built on TailAdmin, a free React and Tailwind CSS admin dashboard template. It provides a comprehensive backend dashboard solution with role-based access control, city management functionality, and various administrative features. The project is actively maintained with recent updates including AuthorizedRoutes components for security, improved sidebar styling, and modern React 19 compatibility.

### Main Function Points
- Role-based access control with AuthorizedRoutes component
- City management functionality with form and table display
- Comprehensive admin dashboard UI with 30+ components
- Data visualization with charts (Line and Bar)
- Profile management and authentication forms
- User-friendly sidebar navigation with dark mode support
- Tables with sorting and filtering capabilities
- Calendar with drag-and-drop functionality
- Alert, dropdown, modal, and button UI elements

### Technology Stack
- React 19
- TypeScript
- Tailwind CSS v4
- Vite (build tool)
- ApexCharts (data visualization)
- flatpickr (date picker)
- ESLint (code quality)
